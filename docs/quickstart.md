# Akademos quickstart

Akademos est un tuteur IA pour enfants de 5 à 17 ans. Parcours d'apprentissage personnalisés, alignés sur le programme français, avec tableau de bord pour les parents.

## Three steps

1. Open [Akademos](https://www.akademos.app)
2. Follow the on-page flow
3. Bookmark https://www.akademos.app for later
